脚本：scripts/nsfw_filter.py
功能：扫描目录中的图片和视频文件，检测NSFW（不良内容）并移动到指定输出目录。

运行方式：
  python scripts/nsfw_filter.py --input-dir <输入目录> --output-images <图片输出目录> --output-videos <视频输出目录>

示例：
  python scripts/nsfw_filter.py --input-dir ./media --output-images ./nsfw_images --output-videos ./nsfw_videos
  python scripts/nsfw_filter.py --input-dir ./media --output-images ./img --output-videos ./vid --dry-run

参数说明：
  --input-dir       输入媒体文件目录（必填）
  --output-images   NSFW图片输出目录（必填）
  --output-videos   NSFW视频输出目录（必填）
  --threshold       检测置信度阈值（见下方各后端推荐值）
  --sample-interval 视频帧采样间隔（秒），默认 2.0
  --max-frames      每个视频最大检测帧数，默认 100
  --dry-run         仅报告检测结果，不移动文件
  --detector        检测后端，可选 nudenet/clip/falconsai，默认 nudenet

各检测后端推荐参数：
  nudenet（默认，最稳定）  --threshold 0.5
  clip（语义检测）         --threshold 0.70   # 必须用较高阈值，否则误报多
  falconsai（轻量二分类）  --threshold 0.5

优化建议：
1. falconsai 继续做第一轮主筛，因为你已经验证它整体效果最好。
2. 对 falconsai 判定为 safe、但分数接近阈值的图片，再让 clip 做第二轮复检，专门捞漏网图。
3. nudenet / OpenCV 在 Windows 下对中文目录、非 ASCII 路径不稳定，脚本已改成自动复制到临时 ASCII 路径后再检测，不需要你再手工改文件名。
